GOAL


The goal is to accurately predict the Length of Stay using IBM Cognos Analytics for each patient on case by case basis so that the Hospitals can use this information for optimal resource allocation and better functioning. The length of stay is divided into 11 different classes ranging from 0-10 days to more than 100 days.
